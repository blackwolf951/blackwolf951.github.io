關於我

研究動機與背景

我在學士階段從生物醫學工程跨入電機工程,累積了醫學背景與電路、韌體及軟體的整合能力.在專題研究「基於 MQTT 與 API 應用於智慧尿布即時監控系統設計」中,我設計了感測器與 ESP32 物聯網平台,並透過 MQTT 與 API 實現多設備即時監控,成功驗證系統的可靠性與延遲特性.

這段經驗使我深刻體會到:若能進一步結合低功耗積體電路設計與人工智慧健康監測模型,不僅能提升臨床照護效率,還能拓展應用至長照與精準健康.因此,我希望在國立臺灣大學精準健康碩士學位學程中,持續深化「智慧感測 + AI」的跨領域研究.

個人優勢

🏥 醫學背景

曾就讀於生物醫學工程學系,修習解剖學、生物學及醫療器材相關課程,建立紮實的醫學基礎,對臨床應用與醫療設備運作原理具備清晰理解.

⚡ 跨域專業

目前主修電機工程學系,修習電路學、電子學、數位邏輯、電腦輔助電路設計以及多門程式語言課程,累積電機與資訊工程的跨領域能力,具備軟硬整合與系統設計的技術基礎.

實務經驗

🔬 醫療器材研發

曾參與醫鼎企業的醫療器材設計,實際接觸醫材開發流程,深入了解臨床需求與工程設計間的連結,培養了以使用者需求為導向的設計思維.

💻 資訊系統維運

在義大開發資訊室任職,負責硬體與軟體的維護,處理百貨公司 POS 系統與網路設備問題,培養了系統整合與問題解決的實務能力.

專題與研究成果

除智慧尿布專題外,我亦參與多項物聯網應用開發,如「即時監控保鮮盒系統」與「智慧尿布監控系統」,並與高醫岡山分院及設備供應商合作,探討系統優化與臨床應用,驗證系統的可靠性與實用性.

此外,大三暑假期間進入義大開發資訊室軟體部門,負責百貨公司 POS 系統維護,並參加綠點子創新競賽榮獲銅牌,展現將理論轉化為實務成果的能力.

研究展望

透過這些學術與實務經驗,我能從臨床需求出發,結合醫學基礎、電機專業與資訊技術,發展具體可行的智慧醫療解決方案.我期望在臺大精準健康碩士學位學程中進一步精進研究,將成果應用於智慧醫療與精準健康領域,推動臺灣醫療科技的創新與發展.
